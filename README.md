# Emertech Blocks WP

Exclusive WordPress plugin developed for the Emertech website that can be accessed [here](https://emertech.pt/).

Based on 
- Object-oriented **PHP** for WordPress
- **JS** along with Babel/**React** components 
- **Bootstrap** for frontend.
